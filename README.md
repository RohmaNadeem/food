# Foodie's Delight Restaurant Website

A modern, responsive restaurant website showcasing the menu, location, and contact information. Built with HTML5, CSS3, and Bootstrap.

## Features

- **Home Page** – hero section with call‑to‑action buttons, featured dishes, about section, and testimonials.
- **Menu Page** – categorized menu (starters, mains, desserts) with pricing, dietary badges (vegetarian, gluten‑free, spicy), and images.
- **Contact Page** – contact form, opening hours, map integration, and location details.
- **Responsive Design** – works seamlessly on desktops, tablets, and mobile devices.
- **Smooth Navigation** – sticky navbar with hover effects and anchor‑based scrolling.
- **Interactive UI** – card hover animations, scroll‑based navbar styling, and a newsletter sign‑up.

## Technologies Used

- HTML5
- CSS3 (custom styles + Bootstrap 5)
- JavaScript (scroll effects, smooth anchor links)
- Font Awesome icons
- Google Fonts

## How to Run

1. Clone the repository:  
   `git clone https://github.com/RohmaNadeem/food.git`
2. Open any `.html` file (e.g., `index.html`) in a web browser.
